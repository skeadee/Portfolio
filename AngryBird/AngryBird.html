    Angry Bird 핵심 요소

    Ray 와 Vector의 이해, 적 생성 로직 구현 및 코드의 재사용

    핵심요소

    Player


    위 동영상의 핵심 기능은
    공을 당기는 범위 설정 , 공이 날라가는 지점 입니다.

    공을 당기는 범위를 구현하기 위해서는
    (마우스의 위치(Vector3) - 공의 처음 위치(Vector3)) = (처음 위치에서 마우스의 방향)Vector3.sqrMagnitude기능을 이용하여 처음 위치에서 마우스가 얼마나 떨어져 있는지의 값을 실수로 받아 사용합니다.
    (sqrMagnitude란?)

    공이 날아가는 지점을 구현하기 구현하기 위해서는
    현재 속도를 비교할 Vector2 ori 선언 후 Vector3.Zero로 초기화를 한 후 Ball의 Rigidbody2D.velocity.sqrMagnitude 값과 ori.sqrMagnitude 값을 서로 비교합니다.
    Ball.velocity 값이 ori.velocity 값보다 크다면 아직 최고 속력이 아님으로 ori = Ball으로 변경하여 다시 값을 변경합니다.
    반대로 ori.velocity 값이 Ball.velocity 보다 크다면 현재 속도는 최대치의 바로 전 단계임으로 Ball = ori로 설정한 뒤 공이 날라갈수 있도록 구현하였습니다.

    적 구조물 생성 로직

    게임 시작 , 재시작시 맵의 랜덤 구성

    게임 실행 시 항상 구조물이 다르게 나오는데
    이를 구현하기 위해 필요한 변수 값(가로의 최대 길이 , 가로 이동 크기 , 세로 이동 크기 등등)을 받아서 게임 마다 적이 랜덤으로 재생성되는 로직을 직접 만들어 보았습니다.
    코드는 별로 어렵지 않은 코드이지만 이런 작은 로직이라도 제가 직접 만들고 수정해 보는 경험이 나중에는 큰 도움이 될거라고 생각합니다.
